# ASAP7 Generated Standard Cell GDS

This repository provides the GDS layouts of ASAP7 standard cells generated by our automatic standard-cell layout generation framework.

The layouts are released to improve reproducibility and to facilitate comparison with future standard-cell layout generation studies.

## Overview

The GDS files in this repository correspond to the standard-cell layouts generated in the paper:

**"Standard Cell Layout Generator Empowered by ILP-based Routing with Dynamic Grid-Shifting for Advanced Nodes"**

The proposed framework generates standard-cell layouts using transistor folding, diffusion-sharing-driven partitioning, quality-aware sorting, SMT-based placement, ILP-based routing, dynamic grid-shifting, sweep-line-based routing acceleration, and I/O pin extension.

All released layouts are generated for the ASAP7 predictive process design kit and are organized into five categories:

- `SEQ`
- `SIMPLE`
- `AO`
- `OA`
- `INVBUF`

## Repository Structure

The repository is organized as follows:

```text
.
├── SEQ/
├── SIMPLE/
├── AO/
├── OA/
└── INVBUF/
```

Each folder contains the generated GDS files for the corresponding cell category.

## Cell Categories

- `SEQ`: sequential cells, such as flip-flops and latch-related cells
- `SIMPLE`: simple combinational cells, such as NAND, NOR, AND, OR, XOR, XNOR, MAJ, FA, and HA cells
- `AO`: AO-type compound logic cells
- `OA`: OA-type compound logic cells
- `INVBUF`: inverters, buffers, and high-buffer cells

## File Format

All layouts are provided in GDS format.

These GDS files can be opened and inspected using layout viewers such as:

- KLayout
- Cadence Virtuoso
- Mentor Calibre DESIGNrev
- Other GDS-compatible layout tools

## Technology

The released layouts target the ASAP7 predictive technology.

Please refer to the official ASAP7 PDK documentation for technology files, layer definitions, and design-rule information.

## Validation

The generated standard-cell layouts were validated using Mentor Calibre for:

- Design Rule Checking (DRC)
- Layout Versus Schematic (LVS)

The released GDS files are intended to support reproducibility of the layout-generation results reported in the paper.

## Notes

- This repository contains generated GDS layouts only.
- Technology files, device models, and ASAP7 PDK files are not included.
- Users should obtain the ASAP7 PDK separately from its official source.
- Some downstream verification or visualization steps may require the corresponding ASAP7 technology setup.
